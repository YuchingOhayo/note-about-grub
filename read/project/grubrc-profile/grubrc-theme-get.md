### fzf

執行下面指令，提供fzf列表選項安裝

``` sh
grubrc-theme-get fzf
```

### all

執行下面指令，安裝以下所有的「theme」

``` sh
grubrc-theme-get all
```

### poly_dark

執行下面指令，單一安裝「theme / [poly_dark](https://github.com/shvchk/poly-dark)」

``` sh
grubrc-theme-get poly_dark
```
