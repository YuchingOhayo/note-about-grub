# clear

學習grub指令「[clear](https://www.gnu.org/software/grub/manual/grub/html_node/clear.html#clear)」。


## 操作範例

在「grub>」命令提示字元下，執行下面指令

``` sh
help clear
```

顯示

```
Usage: clear
Clear the Screen.
```

接著執行

``` sh
clear
```

就會把畫面清空，只留下一個命令提示字元「grub>」
