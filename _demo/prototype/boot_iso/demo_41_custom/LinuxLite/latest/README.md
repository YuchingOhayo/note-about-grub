

# Demo Grub Boot ISO / LinuxLite

* /etc/grub.d/41_custom

## Menu Entry

| Title | Iso File Path |
| --- | --- |

``` sh
http://repo.linuxliteos.com/linuxlite/isos/5.8/linux-lite-5.8-64bit.iso
```


## iso-download.sh

``` sh
wget -c -i iso-download.txt
```
